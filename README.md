# china-city-data
中国城市数据（首字母排序，json文件格式）


## install

```shell
npm install china-city-data
```


## 数据格式(json)

```js

{
    A: [
        {
            "id": "152900", //城市编码
            "name": "\u963f\u62c9\u5584\u76df"
        }
        // ……
    ],
    // ……
    Z: [
        // ……
    ],
    // 热门城市
    hot: [
        {
            "id": "110100",
            "name": "\u5317\u4eac"
        }
    ]
}

```
